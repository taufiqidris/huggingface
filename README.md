# GitHub to Hugging Face Demo

This is a minimal Hugging Face Space project for classroom deployment.

## Files
- `app.py` — Gradio app
- `requirements.txt` — Python dependencies
- `README.md` — Hugging Face Space config + documentation
- `.gitignore` — Ignore local junk files

## Run locally
```bash
pip install -r requirements.txt
python app.py
```

## Deploy
1. Create a new Hugging Face Space
2. Choose **Gradio**
3. Upload these files or push them from GitHub
4. Wait for the build to finish

## Why this version
- Lightweight
- Free-tier friendly
- Fast to build
- Good for student demos
